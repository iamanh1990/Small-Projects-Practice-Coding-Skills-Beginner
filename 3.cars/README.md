# Cars Generator

I this small project, I used vanila JS to practice DOM manipulation and form input manipulation. The users can input their cars credentials and then search for their cars using several simple functions.

## Technologies used.

Built with:

- HTML
- CSS
- JavaScript

## Setup and Usage

Everything is setup, double click on index.html to open the file on your local machine. This is just a small project so I do not want to deploy such a static website.
